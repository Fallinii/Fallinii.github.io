3.分割线

三个或三个以上的-或者*,行内不能有其他东西，星号和减号之间可以插入空格

示例：

4.字体

把符号添加到文字两端,或使用HTML标签

示例：
  • 加粗 **
    加粗的文字
  • 斜体 *
    斜体的文字
  • 斜体加粗 ***
    斜体加粗的文字
  • 删除线 ~~
    加删除线的文字 （MarkdownPad2不支持，博客支持，可以用）
    使用HTML<del>标签
    删除线
  • 下划线 <u>标签
    下划线的文字

5.列表

无序列表

列表内容前添加 - / + / *,这些标记后要添加一个空格

示例：
  • 列表内容1
  • 列表内容2
  • 列表内容3

有序列表

列表内容前数字加点 1.

示例：
  1. 列表内容1
  2. 列表内容2
  3. 列表内容3

列表嵌套

下一级列表前面敲三个空格

示例：
  • 一级无序列表
    • 二级无序列表
    • 二级无序列表
  • 一级无序列表
    1. 二级有序列表
    2. 二级有序列表
  1. 一级有序列表
    • 二级无序列表
    • 二级无序列表
  2. 一级有序列表
    1. 二级有序列表
    2. 二级有序列表

6.引用

引用的文字前加>, 后面跟一个空格

示例：

引用的内容
同一层

下一层引用

可以嵌套好多层

引用中使用列表

在>空格后直接加-或者1.空格

引用中使用列表

  • 无序第一项
  • 无序第二项
  1. 有序第一项
  2. 有序第二项

列表中使用引用

列表内的>前添加四个空格或一个Tab键

示例：
  • 第一项

    Markdown
    简单易学

  • 第二项

7.图片

语法：！[图片alt](图片地址 "title(可选)"）

示例：

还不会搞图片地址

8.超链接

语法：[超链接名](超链接地址 "title(可选)") 或者 <链接地址>

示例：

百度
http://baidu.com

9.代码

单行代码：用反引号(`)包起来

示例：

console.log("hello world")

代码块： 整体代码块前添加四个空格或一个Tab键,与上一个段落留出一个空行

示例：
function foo() {  
    console.log("")  
}

10.表格

MarkdownPad2免费版不支持表格语法，需切换到extra版

语法：

| 表头 | 表头  
| ---- | ----  
| 单元格 | 单元格
表头 表头
单元格 单元格

11. 支持的HTML元素

目前支持的元素有 <kbd> <b> <i> <em> <sup> <sub> <br> <u> 等

示例：

键盘：ctrl
加粗：black
基于样式的斜体：italian
基于内容的斜体：element
上角标：20481
下角标：20482
换行：br

下划线： under

12. 转义字符

反斜杠\转义特殊字符

示例：

文本加粗
**正常显示星号** 

支持的符号
\ 反斜杠  ` 反引号  * 星号  _ 下划线  {} 花括号  [] 方括号
() 小括号  # 井字号  + 加号  - 减号  . 英文句号  ! 感叹号
